## Featured projects

### 🩺 Nursing Care — enterprise on-demand care platform
A three-tier system for managing on-demand nursing and home residential care services.
- [`NursingCare`](https://github.com/sterlingdiazd/NursingCare) — .NET 8 backend, Clean Architecture + DDD, JWT/OAuth2, EF Core, CQRS handlers
- [`nursing-care-web`](https://github.com/sterlingdiazd/nursing-care-web) — React + TypeScript admin dashboard with audit logs
- [`nursing-care-mobile`](https://github.com/sterlingdiazd/nursing-care-mobile) — React Native + Expo app for nurses and admins

### 🤖 AI Billing Validation Agent
Multi-agent AI pipeline that detects overbilling, rate mismatches, and contract violations before invoices are sent. Anthropic Claude + OpenAI multi-provider support, deterministic fallback, schema-versioned JSON contracts, kill switch, audit log, RBAC role model, 42 unit + E2E tests, Streamlit UI.
→ Repo on org: [`TP-EvaluationProject-AITransformation`](https://github.com/AI-Transformation-Technical-TestPack/TP-EvaluationProject-AITransformation)

### 🔎 Job Search Automation Tool
Python automation that drives LinkedIn via Playwright + Chrome DevTools Protocol — keyword-driven search, Easy-Apply filtering, incremental Excel logging, structured retry/error handling.
→ [`job-search-automation-tool`](https://github.com/sterlingdiazd/job-search-automation-tool)

### 📝 Form Validation App
Compact React + modal-driven form validation reference. Useful as a teaching artifact.
→ [`form-validation-app`](https://github.com/sterlingdiazd/form-validation-app)
